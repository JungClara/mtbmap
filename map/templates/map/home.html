We offer free map for mountain biking (MTB) and hiking, which is based on data of the project OpenStreetMap. These data are consistently improved by keen contributors. They add new routes, points of interest and other objects. We update these contributions normally once a week and they are reflected in MTB map with some delay. Map export in corresponding tab is done with our latest data. That is how you can get up-to-date map in printable resolution.

Legend is displayed for current map zoom and is available only for MTB layer. You can use routing functionality and create altitude profiles in the tab Routes.

In case of any questions, remarks or new ideas, don't hesitate to send us an email. Address is in the bottom left corner.

Other details and tutorials, which help you contribute to OpenStreetMap project are on our OpenStreetMap Wiki page. Developers can look at our GitHub project pages.
